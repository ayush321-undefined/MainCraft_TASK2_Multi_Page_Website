## ✨ Overview

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=rect&color=6366F1,22D3EE&height=3&width=100%" width="100%"/>
</div>

<br/>

**Nexus** is a premium, fully responsive SaaS landing platform built with vanilla HTML, CSS, and JavaScript. It showcases a modern infrastructure product with dark/light theming, smooth animations, and a polished multi-page experience — all without a single framework dependency.

Founded in 2014, Nexus was born from frustration with the complexity of modern cloud infrastructure. From a Bash script in a San Francisco basement to an orchestration platform processing **2.4 million daily API requests** across 80 countries — this is that story.

```
✦ Now in Public Beta  ·  50,000+ Active Users  ·  Bootstrapped & Profitable
```

## 🏁 Getting Started

### Prerequisites

No build tools, no package manager, no dependencies. Just a browser.

### Option 1 — Live Server (Recommended)

```bash
# Clone the repository
git clone https://github.com/your-username/nexus.git
cd nexus

# If you have VS Code, install the Live Server extension
# Right-click index.html → "Open with Live Server"
```

### Option 2 — Python HTTP Server

```bash
cd nexus

# Python 3
python -m http.server 8080

# Then open → http://localhost:8080
```

### Option 3 — Node HTTP Server

```bash
cd nexus

# Install serve globally (one-time)
npm install -g serve

# Start server
serve .

# Then open → http://localhost:3000
```

> ⚠️ **Important:** Do **not** open HTML files directly via `file:///` in your browser.  
> Browsers block local file navigation for security reasons (`file:` URLs are treated as unique origins).  
> Always use a local server (any option above works).
